# 📌 Project Overview

In this project, I used Python libraries such as Pandas, Matplotlib, and Seaborn
to perform Exploratory Data Analysis (EDA), data cleaning, and statistical analysis.
The SMART method was applied to define a structured problem statement and set a
measurable sales target of 100 units within 6 months. Hypothesis testing was also
conducted to compare the average price of electric versus fuel-based vehicles.
Final results and insights are visualized using Tableau.

## 1. Exploratory Data Analysis (EDA)

- Performing data exploration and analysis using Python libraries such as Pandas
to uncover patterns, trends, and insights from the used car stock data.

## 2. Business Analysis with SMART Method

- Defining a structured problem statement using the SMART framework to improve
used car sales performance, accelerate stock turnover, and increase consumer
appeal with a target of 100 units sold within 6 months.

## 3. Data Visualization

- Creating interactive and insightful data visualization dashboards using
Tableau to present stock conditions and findings in a clear and actionable format.

## 4. Statistical & Hypothesis Testing

- Conducting hypothesis testing (Two Sample Independent T-Test) to analyze
whether there is a significant difference in average price between electric
and fuel-based vehicles.
